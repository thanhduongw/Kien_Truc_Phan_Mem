# Frontend - Travel Booking

This is a lightweight React UI that only calls the Orchestrator service.

## Features
- Login
- View tour list
- View tour detail
- Book a tour

## Configure
Create a `.env` file if you need to point to a LAN host:

```
VITE_ORCHESTRATOR_URL=http://localhost:8080
```

## Run locally
```
npm install
npm run dev
```
